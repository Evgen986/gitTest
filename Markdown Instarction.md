## Списки

Чтобы добавить ненумерованные списки необходим пункты выделить звездочкой (*). Например, вот так:
* элемент 1
- элемент 2
* элемент 3

Чтобы добавить нумерованные списки, необходимо пункты пронумеровать. Например, вот так:
1. Первый пункт
2. Второй пункт
